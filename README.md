## Features
- **Interactive Maps** - Using Leaflet and It's plugins to build some features for interactive maps.
- **Basemap Options** - Two options for viewing the maps with OSM maps and ESRI Basemaps (Imagery)
- **Year Slider** - Easily view the projects locations through scrolling the year
- **Informative Popups** - Popups that contain short information about my projects
- **Portofolio and Contacts Button** - Easy Access for anyone who interesting to know some details about myself.
